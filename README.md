# ü¶† Malaria Diagnosis Using Deep Learning

This project implements an end-to-end deep learning pipeline for automatic malaria diagnosis using microscopic blood cell images. Leveraging the TensorFlow Malaria Dataset, it trains a convolutional neural network (CNN) to classify images as Parasitized or Uninfected. Automated detection of malaria-infected blood cells using TensorFlow, NumPy, Matplotlib, and Seaborn.


**The system uses:**

    TensorFlow for model development
    
    NumPy for preprocessing and numerical operations
    
    Matplotlib & Seaborn for data visualization and performance analysis
    
    This project demonstrates how AI can support disease diagnosis by providing fast, reliable, and scalable image-based screening.


## üîç Data Handling

    Imports and preprocesses the TensorFlow Malaria Dataset
    
    Normalizes images and prepares train/validation datasets
    
    Includes class balancing and shuffling

## üß† Deep Learning Model

    CNN architecture built with TensorFlow/Keras
    
    Trained to distinguish parasitized vs. uninfected cells
    
    Tracks accuracy, loss, and validation performance

## üìà Model Evaluation

    Confusion matrix
